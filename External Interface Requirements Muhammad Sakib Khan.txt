<h1>3. External Interface Requirements</h1>
<h2>3.1 User Interfaces </h2>
<p>The website will have a straightforward and a clean look which easy to navigate. At first sight, a user will feel everything in the website is well organized and easy to explore. There will be two separate User Interfaces. One for the teacher while other for the student.</p>
<p>There will be graphical user interface (GUI) which will allow the users to provide feedback. The teachers can post grades here on real time and system will make a statistical standing and keep it updated. Errors will be displayed if user give wrong inputs.  There will also be a help button which when the user presses will show information to the user about the page content and this button will appear on every page. </p>
<p>Front end Software: HTML, CSS and WordPress</p>
<p>Back end Software: PHP, Laravel Framework & MySQL</p>
<h2>3.2 Hardware Interfaces</h2>
<p>Any Device that is able to run a web browser which has the ability to process above mentioned Front end software and also the device has to meet the requirements of the browser to run without lagging. </p>
